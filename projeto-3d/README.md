# projeto-3d
Maquete 3D interativa da obra — Mariára Rita Siqueira Ornellas

Visualização em Three.js do projeto arquitetônico aprovado (Av. Projetada,
Quadra H, Lote 38, Residential Village, Campos dos Goytacazes/RJ), com
acompanhamento do andamento da construção por etapas (fundação, alvenaria
por pavimento, cobertura, acabamento). O progresso marcado fica salvo
localmente no navegador de quem acessa.
